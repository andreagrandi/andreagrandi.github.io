Using Python ipdb from Jupyter

Tue 10 May 2016 | in Development, Python | tags: debuggingipdbPython

If we try to use the usual ipdb commands from a Jupyter (IPython notebook)

``` {.lang:python .decode:true} import ipdb; ipdb.set_trace()

we will get a similar error:

``` {.lang:python .highlight:0 .decode:true}

      5 print b

The solution is to use Tracer instead:

{.lang:python .decode:true} from IPython.core.debugger import Tracer Tracer()()
